# AnKor - Android & Kotlin Learning App

Современное React приложение для изучения Android и Kotlin с поддержкой PWA и темной темы.

## 🚀 Технологии

- **React 18** - Современная библиотека для UI
- **TypeScript** - Типизация и лучший DX
- **Material-UI** - Компоненты и темы
- **Zustand** - Управление состоянием
- **React Router** - Маршрутизация
- **Framer Motion** - Анимации
- **Vite** - Быстрая сборка
- **PWA** - Прогрессивное веб-приложение

## ✨ Особенности

### **Темная тема**
- Полностью черные градиенты для темной темы
- Автоматическое переключение фона
- Настраиваемые цветовые схемы

### **PWA поддержка**
- Установка как приложение
- Офлайн работа
- Адаптивный дизайн

### **API интеграция**
- Работа с сервером
- Офлайн синхронизация
- Автоматическое сохранение

## 📦 Установка

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Запуск сервера
npm run server:dev

# Запуск всего проекта
npm run dev:full
```

## 🛠 Скрипты

- `npm run dev` - Запуск dev сервера
- `npm run build` - Сборка для продакшена
- `npm run preview` - Предварительный просмотр
- `npm run server:dev` - Запуск API сервера
- `npm run dev:full` - Запуск клиента и сервера
