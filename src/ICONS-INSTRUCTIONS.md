# 🎨 Создание иконок для AnKor

## Быстрый способ:

### 1. Откройте create-icons.html
- Откройте файл `create-icons.html` в браузере
- Нажмите кнопку "Скачать иконки"
- Сохраните файлы в папку проекта

### 2. Альтернативный способ - онлайн генератор:
1. Перейдите на https://favicon.io/favicon-generator/
2. Введите текст "A" 
3. Выберите цвет фона: #3b82f6 (синий)
4. Выберите цвет текста: белый
5. Скачайте иконки
6. Переименуйте в `icon-192.png` и `icon-512.png`

### 3. Ручное создание:
- Создайте квадратные изображения 192x192 и 512x512 пикселей
- Синий фон (#3b82f6) с белой буквой "A" по центру
- Сохраните как PNG файлы

## После создания иконок:
1. Поместите файлы в корневую папку проекта
2. Перезапустите сервер
3. Откройте приложение в Safari на iPhone
4. Добавьте на главный экран
