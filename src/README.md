# AnKor - Приложение для изучения вопросов собеседований

Современное веб-приложение для подготовки к собеседованиям с помощью флеш-карт и тестов.

**Разработчик:** Bakhaev Rizvan

## 🚀 Возможности

- **Два режима изучения:**
  - **Режим изучения** - классические флеш-карты с вопросами и ответами
  - **Режим тестирования** - выбор правильного ответа из нескольких вариантов

- **Статистика прогресса:**
  - Общее количество вопросов
  - Количество изученных вопросов
  - Количество правильных ответов

- **Современный дизайн:**
  - Mobile First подход
  - Адаптивный дизайн для всех устройств
  - Красивые анимации и переходы
  - Темная/светлая тема

## 📱 Как использовать

1. **Откройте `index.html`** в любом современном браузере
2. **Добавьте первый вопрос** нажав кнопку "Добавить вопрос"
3. **Выберите режим изучения:**
   - **Изучение** - для запоминания с помощью флеш-карт
   - **Тест** - для проверки знаний
4. **Изучайте вопросы** и отмечайте правильные/неправильные ответы
5. **Следите за прогрессом** в статистике

## 🎯 Режимы работы

### Режим изучения
- Показывает вопрос
- По клику "Показать ответ" открывается правильный ответ
- Можно отметить ответ как правильный или неправильный
- Статистика обновляется автоматически

### Режим тестирования
- Показывает вопрос с несколькими вариантами ответов
- Один правильный ответ + 2 неправильных (из других вопросов)
- После выбора ответа показывается правильный вариант
- Автоматический переход к следующему вопросу

## 💾 Сохранение данных

Все данные сохраняются в локальном хранилище браузера (localStorage):
- Вопросы и ответы
- Статистика изучения
- Настройки приложения

Данные сохраняются автоматически и не теряются при закрытии браузера.

## 🔧 Настройка

Приложение готово к использованию без дополнительной настройки. Просто откройте `index.html` в браузере.

Для разработки рекомендуется использовать локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```
